
# laravel-nuxt-myattheingiaung

## Git clone

git clone https://github.com/scm-ojt/laravel-nuxt-myattheingiaung.git

## Laravel Set-up

- cd api
- copy env.example .env
- env setup
- create database 
- composer install

## Nuxt Set-up

- cd client
- npm install

## Laravel Run

- cd api
- php artisan serve
- php artisan migrate
- php artisan db:seed

## Nuxt Run

- cd client
- npm run dev
