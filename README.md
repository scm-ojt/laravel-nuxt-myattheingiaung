# laravel-nuxt-myattheingiaung

## Git clone

git clone https://github.com/scm-ojt/laravel-nuxt-myattheingiaung.git

## Set-up

- copy env.example .env
- env setup
- create database 
- npm install
- composer install

## Run

- npm run dev
- php artisan serve
- php artisan migrate
- php artisan db:seed
